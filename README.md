# FC-N28 Level 5 3D Campus Tour

## Project Description
This project is an interactive 3D campus tour of FC-N28 Level 5, focusing on the elevator, corridor, Computer Programming Lab, and exit door area.

## Technologies Used
- Three.js
- Vite
- JavaScript
- HTML
- CSS
- GLB 3D models

## Features
- First-person navigation
- Interactive lift door, lab door, and exit door
- Automatic information panel
- Minimap and floor plan overlay
- Day/night lighting mode
- Audio effects
- Collision detection

## How to Run
npm install
npm run dev

## Build
npm run build
